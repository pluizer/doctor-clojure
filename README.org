#+TITLE: doctor-clojure

Check and auto-fix unbalanced delimiters in Clojure source files.

** Install

#+begin_src sh
curl -LO https://raw.githubusercontent.com/pluizer/doctor-clojure/master/doctor-clojure.sh
chmod +x doctor-clojure.sh
mv doctor-clojure.sh ~/.local/bin/doctor-clojure
#+end_src

Requires Clojure CLI.

** Usage

#+begin_src sh
doctor-clojure <file>
doctor-clojure check <file>
doctor-clojure fix <file>
doctor-clojure repair <file>
doctor-clojure --help
#+end_src
